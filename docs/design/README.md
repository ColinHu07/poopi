# Poopi visual direction

`poopi-ui-direction-v1.png` is the product reference for Poopi's July 2026 redesign. It is intentionally a direction board rather than a pixel-for-pixel screen specification.

The implemented system carries forward:

- warm paper backgrounds with espresso text;
- jade for trustworthy actions and coral for contribution actions;
- tactile rounded cards, pills, and bottom sheets;
- bold editorial headings with compact, plain-language supporting copy;
- the Poopi mascot as a playful brand accent without turning the interface into a novelty UI;
- prominent map, status, directions, review, and edit-review actions.

The concept was generated from Poopi's existing mascot using the prompt: “Create a polished mobile product UI direction board for Poopi, a playful bathroom discovery and rating app. Show four cohesive phone screens: nearby map, Wenwen bathroom details with Edit your review, good/bad bathroom rating labels, and profile review history. Use warm cream paper, espresso brown, cocoa, jade, coral, butter yellow, and mint; chunky editorial typography; tactile rounded cards; simple friendly icons; no gradients, glassmorphism, purple AI aesthetic, or unnecessary dashboard chrome. Keep it fun, legible, practical, and accessible.”
